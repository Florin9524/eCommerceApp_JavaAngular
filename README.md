"# E-Commerce APP in Java & Angular" 

Full Stack Application for skill development
Front-End: Angular
Back-End : Spring Boot
Database support with FULL CRUD: Create, Read, Update, Delete 

Rest API -> Representational State Transfer

Requirments:
First Stage: 
* Show a list of products
* Add products to shopping cart
* Shopping cart check out
* User Login/Logout security
* Track Previous orders for logged in users


Plan
Step 1: setup Database tables
Step 2: Create Spring Boot starter project
    spring-boot-starter-data-jpa
    spring-boot-starter-data-rest
    mysql-connector-java
    lombok
Step 3: Create Product and ProductCatergory tables
Step 4: Create REST APIs with Spring Data JPA Repositories and Spring Data REST
